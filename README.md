# BlackWidowV3mini-diy

重新设计了BlackWidowV3mini的外壳，使用了gasket结构，外部无螺丝，窄边框。
重新绘制了矩阵pcb，单键开槽，热插拔。



## tips
1. 模型可直接用于3D打印，如要cnc的话需要自己更改外壳的热压螺母孔为m1螺纹孔，且重新考虑天线摆放位置。
2. 3d打印上盖可能因为有长条结构出现变形，建议多打印几个组装时选择相对完美的使用。
